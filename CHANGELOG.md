# Changelog

All notable changes to Seeva AI Assistant will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Auto-update functionality with signature verification
- GitHub Actions CI/CD workflows for automated releases
- Multi-platform build support (macOS, Linux, Windows)
- Update checker UI in settings modal
- macOS code signing and notarization support

## [0.1.0] - 2025-11-11

### Added
- Initial release of Seeva AI Assistant
- Chat interface with thread management
- Screenshot capture integration
- Claude AI model support (Sonnet 4.5, Haiku 4.5, Opus 4.1)
- Global keyboard shortcut for quick access
- Dark mode support
- Persistent chat history with SQLite
- API key management
- Model selection in settings
- Always-on-top floating window
- Transparent window design with glass effects
